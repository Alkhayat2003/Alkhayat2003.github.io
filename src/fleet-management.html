Fleet Management Project

Advanced system for real-time vehicle tracking. I built the entire backend to ensure accurate and efficient tracking.

Features

  • Real-time vehicle tracking using GPS
  • Vehicle location display on map
  • Route, time, and speed reporting
  • Geofencing alerts for entering/exiting zones
  • Driver and vehicle data management
  • API for system integration

Technologies Used

  • Python (Primary programming language)
  • Django (Web framework)
  • Django REST Framework (API building)
  • MySQL (Database)
  • WebSocket (Real-time communication)
  • Git (Version control)
