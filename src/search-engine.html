Search Engine Project

Part of a search engine capable of reading documents, indexing data, and managing them effectively. I focused on building the backend and integrating advanced indexing technologies.

Features

  • Reading and parsing various document formats
  • Efficient data indexing using advanced techniques
  • Fast and accurate search within indexed data
  • Management and organization of indexed documents
  • API interface for interacting with the search engine

Technologies Used

  • Python (Primary programming language)
  • Django (Web framework)
  • Elasticsearch (Search and analytics engine)
  • REST APIs (For component communication)
  • Git (Version control)
Python Django Elasticsearch REST API Git

My Role

  • Built the entire backend infrastructure
  • Developed document reading and parsing mechanisms
  • Managed integration between Django and Elasticsearch
  • Set up and managed data indexes
  • Designed and implemented APIs
  • Tested performance and indexing effectiveness
