Weather Application

Application allowing users to add their favorite cities and inquire about the weather in different cities around the world.

Features

  • Current weather information for all cities
  • Ability to add cities to favorites list
  • Display detailed weather (temperature, humidity, wind speed, etc.)
  • Attractive and user-friendly interface
  • Quick city search
  • Save user preferences

Technologies Used

  • Python (Primary programming language)
  • Django (Web framework)
  • Weather API (e.g., OpenWeatherMap)
  • SQLite (Local database)
  • HTML/CSS/JavaScript (Frontend)
  • Git (Version control)
